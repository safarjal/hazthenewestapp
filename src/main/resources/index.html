Haidh and Istihadha Calculatori

Please enter the start date-time for first dam in the first box, and the end date-time for that dam in the second box. To add another period after that, press (➕). If you need to remove a period in the middle, click the remove (❌) button next to it. To add a spot, enter a period where the start time and the end time are the same. Once all periods have been added, click Calculate button, to get the solution.
